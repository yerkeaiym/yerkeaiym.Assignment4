## Installation

PyPI
```bash
pip install request
pip install flask
pip install jwt
```
or from source
```bash
request - https://pypi.org/project/requests/
jwt - https://pyjwt.readthedocs.io/en/stable/
flask - https://flask.palletsprojects.com/en/2.0.x/
```
